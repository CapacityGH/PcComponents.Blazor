# PC Components Blazor

Frontend part of a full-stack web application for managing PC components.

Built with **Blazor** and consumes a RESTful **ASP.NET Core Web API** backend.

---

## Tech Stack

- Blazor
- .NET
- Bootstrap

---

## Features

- View PC components list
- Create and edit components
- Delete components
- Category selection
- API integration via HTTP clients

---

## Notes

- Backend API URL is configured in `appsettings.json`
- This repository contains only the frontend part of the application
